[youtu.be/vB9NTjRyBzY](https://youtu.be/vB9NTjRyBzY)

## Prerequisites
Python

## Getting Started Locally

* Clone the Repo
* install necessary python packages
```sh
pip install -r requirements.txt
```

* Run the Web Interface Locally:
```sh
streamlit run app.py
```
